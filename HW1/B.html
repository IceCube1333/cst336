In computer graphics software, the distinction between 2D and 3D
is occasionally blurred; 2D applications may use 3D techniques to
achieve effects such as lighting, and primarily 3D may use
2D rendering techniques.

3D computer graphics are the same as 3D models. The model is contained
within the graphical data file. Due to printing, 3D models are not only confined
to virtual space. 3D rendering is how a model can be displayed. Also can be used
in non-graphical computer simulations and calculations.
